# Ratary — Cursor plugin (MCP)

MCP-only Cursor plugin for **Ratary Memory MCP** (`ratary`).

## Install (marketplace)

After listing on [Cursor Marketplace](https://cursor.com/marketplace):

```text
/add-plugin ratary
```

## Configure

Set `RATARY_API_KEY` (`aic_...`) from [Ratary Studio](https://studio.ontorata.com) or your self-hosted instance.

Default REST base: `https://ratary.ontorata.com`.

## Full stdio (28 tools)

Clone [ontorata/ratary](https://github.com/ontorata/ratary), configure SQL provider, run `npm run setup` — see [docs/install/cursor.md](../../docs/install/cursor.md).
